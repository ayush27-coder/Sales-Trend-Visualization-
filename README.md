# Sales Trend Visualization

## Objective
Analyze sales data and identify business trends using Tableau.

## Tools Used
- Tableau
- CSV Dataset

## Dashboard Features
- Sales by Category
- Profit by Category
- Sales by Region
- Top Orders by Sales
- Discount vs Profit Analysis

## Key Insights
- South region generated highest sales.
- Technology category earned highest profit.
- Furniture category generated highest sales.
